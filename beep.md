# Beep #

Make some noise to draw the attention of the user.

## Syntax ##
```
Beep,<Action>
```

The selected **Action** can be one of the following values
  * Ok
  * Error
  * Asterisk
  * Confirmation

The sounds produced by each **Action** will vary based on the sound profile of the users computer.

## Example ##
```
beep,ok 
```
