  * [FILENAME / PATH / EXT](strformatFILENAME.md)
  * [INC / DEC / MULT / DIV](strformatINC.md)
  * [LEFT / RIGHT](strformatLEFT.md)
  * [LEN](strformatLEN.md)
  * [LTRIM / RTRIM / CTRIM / NTRIM](strformatLTRIM.md)
  * [POS](strformatPOS.md)
  * [REPLACE / REPLACEX](strformatREPLACE.md)
  * [SHORTPATH / LONGPATH](strformatSHORTPATH.md)
  * [SPLIT](strformatSPLIT.md)
